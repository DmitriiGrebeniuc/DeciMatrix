# DeciMatrix Assets

This folder stores visual assets for the DeciMatrix mobile app.

- `branding/logo/` - logos and logo mark files.
- `branding/app-icon/` - system app icons for Expo.
- `illustrations/` - illustrations for empty states, result screens, and other UI moments.
- `icons/` - small UI icons.
- `images/screenshots/` - application screenshots.
- `images/mockups/` - mockups and presentation images.
